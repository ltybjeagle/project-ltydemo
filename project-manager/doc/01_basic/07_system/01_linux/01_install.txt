Linux系统安装：系统版本（CentOS6.5）
1、vmware tools安装（使用root用户）：
    [root@localhost ~]# cd /
    [root@localhost /]# ls
    [root@localhost /]# cd /media
    [root@localhost media]# ls
    [root@localhost media]# cd VMware\ Tools/
    [root@localhost VMware Tools]# ls
    [root@localhost VMware Tools]# cp VMwareTools-8.1.3-203739.tar.gz    /tmp
    [root@localhost VMware Tools]# cd /tmp
    [root@localhost tmp]# ls
    [root@localhost tmp]# tar -zxf VMwareTools-8.1.3-203739.tar.gz
    [root@localhost tmp]# ls
    [root@localhost tmp]# cd vmware-tools-distrib/
    [root@localhost vmware-tools-distrib]# ls
    [root@localhost vmware-tools-distrib]# ./vmware-install.pl
2、网络设置：
    [root@localhost ~]# ifconfig   查看网卡设置
    [root@localhost ~]# vi /etc/sysconfig/network-scripts/ifcfg-eth0  编辑网卡的配置文件
    添加以下内容：
    IPADDR=本地IP地址
    NETMASK=子网掩码
    GATEWAY=网关
    修改内容：
    ONBOOT=no更改为ONBOOT=yes
    BOOTPROTO=dhcp更改为BOOTPROTO=none
    [root@localhost ~]# service network restart   重启网卡
3、其他设置：
    网络适配器选择：桥接
    虚拟网络编辑器：选择桥接（设置桥接到连接网络的网卡上）
    虚拟机网卡DNS设置成宿主机网卡DNS
4、防火墙开放端口：
    [root@localhost ~]# vi /etc/sysconfig/iptables
    添加端口：-A INPUT -m state -state NEW -m tcp -p tcp -dport 80 -j ACCEPT
    [root@localhost ~]# /etc/init.d/iptables restart       （重启防火墙）

    注意：
    centos7中的防火墙改成了firewall，使用iptables无作用，开放端口的方法如下：
    firewall-cmd --zone=public --add-port=80/tcp --permanent             返回success为成功
    重启防火墙：systemctl restart firewalld.service 
5、常遇问题：
    a、执行shell脚本报错：Permission denied
        没有权限，执行命令chmod 777 service-user.sh
