连接问题：
1、链接redis时只本地localhost(127.0.0.1）链接，不能网络ip(192.168..)链接，如果用网络ip链接会报以下的错误：
	DENIED Redis is running in protected mode because protected mode is enabl
	处理：
	a、打开配置文件把下面对应的注释掉 
		# bind 127.0.0.1 
	b、Redis默认不是以守护进程的方式运行，可以通过该配置项修改，使用yes启用守护进程，设置为no
		daemonize no
	c、保护模式
		protected-mode no 
	d、启动的时候指定配置文件
		./redis-server redis.conf
